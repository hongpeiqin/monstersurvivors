## 已实现

- 设置面板中的粒子特效开关。
- 关闭粒子后立即清空现有粒子，并停止后续生成、物理更新和绘制。
- 粒子开关通过 `localStorage` 持久化。
- 怪物 AI 与群体避让按帧轮转；移动、攻击与受击判定保持逐帧。
- 粒子物理分片更新并做时间补偿。
- 空间哈希网格、动态粒子预算、视口裁剪和自适应 DPR。
- 根据帧耗时自动切换轻载、均衡、降载三档。
