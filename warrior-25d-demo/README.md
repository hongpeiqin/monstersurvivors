# 2.5D 八方向战士 Demo

在线地址：<https://hongpeiqin.github.io/monstersurvivors/warrior-25d-demo/>

这是一个单页 HTML5 Canvas 战斗演示，包含八方向持刀战士、虚拟摇杆、普通攻击与三个技能、持续刷新的追踪怪物、程序化背景音乐和攻击音效。

本版本新增：

- 设置面板中的粒子特效开关，选择会持久化保存。
- 怪物 AI 与群体避让按 3 至 5 帧轮转。
- 粒子物理按 2 至 4 帧轮转，并通过位置外推维持连续运动。
- 自适应轻载、均衡、降载三级调度。
- 粒子生成概率、绘制预算、辉光强度和动态上限控制。
- 地表离屏缓存、空间网格邻居查询和视口裁剪。
- 背景音乐、攻击音效独立音量以及总静音开关。

`index.html` 是 Pages 启动器，`payload/` 中的文本分片是原始单文件 Demo 的 Base64 编码，`manifest.json` 记录构建版本和源码 SHA-256。分片仅用于绕开文件接口上传限制，浏览器载入后执行的内容与独立版 `index.html` 完全一致。
