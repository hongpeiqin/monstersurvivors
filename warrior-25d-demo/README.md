# 霜刃战场 · 2.5D 八方向战士 Demo

在线地址：<https://hongpeiqin.github.io/monstersurvivors/warrior-25d-demo/>

纯 HTML5 Canvas Demo，不依赖第三方运行库。主角支持八方向移动、持刀普通攻击、裂空斩、旋风斩和突进；怪物会持续生成并追踪主角。

## 本次性能实现

- 设置中的“粒子特效”是实际运行开关：关闭后停止粒子生成、物理更新与绘制，并立即清空已有粒子；设置通过 `localStorage` 持久化。
- 怪物 AI 与群体避让按 3 / 4 / 6 帧轮转，移动、攻击与受击判定仍逐帧执行。
- 粒子物理按 2 / 3 / 4 帧轮转，并按最后更新时间补偿位移。
- 空间哈希网格替代怪物全量两两遍历。
- 按帧耗时 EMA 自动切换轻载、均衡、降载三个档位，动态调整粒子上限、绘制预算、生成概率、怪物上限与 DPR。
- 粒子定期压缩、视口裁剪、敌人视口裁剪和 HUD 降频刷新。

## 操作

- 移动：WASD、方向键或左侧八方向摇杆
- 普通攻击：J、空格或右下攻击按钮，可持续按住连击
- 技能：Q 裂空斩、E 旋风斩、R 突进
- M：静音
- V：显示调度调试信息

浏览器控制台可通过 `window.__WARRIOR_DEMO__` 查看运行状态或执行压力测试。
