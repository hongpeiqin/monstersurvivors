# NEON STRIKE V2

单文件 Canvas 飞机大战 Demo，无第三方运行依赖。

## 功能

- Web Audio API 程序化动态配乐与战斗音效
- 音乐、音效独立音量与独立静音
- 总音频静音快捷键与设置持久化
- 可选震动反馈：受击、拾取 Buff、星爆脉冲
- 可选界面抖动：爆炸、受击、Boss 登场与大招冲击均可关闭
- 自适应粒子调度：根据帧耗时和粒子负载，在 1–4 帧间分摊更新
- 粒子上限与动态生成密度：高负载时减少非关键粒子，保留闪光与能量核心
- 移动设备限制高分屏渲染倍率，降低全屏 Canvas 填充压力
- MULTI SHOT：弹幕由 2 发扩展至 9 发
- PARTICLE POWER：子弹伤害最高提升至 2.48 倍
- FIRE RATE：射速最高提升至约 1.72 倍
- 普通波次、Boss 战、战斗核心掉落与粒子打击反馈

## 操作

- WASD / 方向键：移动
- 鼠标 / 触摸：跟随移动
- 空格：星爆脉冲
- 鼠标双击：星爆脉冲
- 手机双击屏幕或双指触碰：星爆脉冲
- 单次触摸：只移动，不消耗大招
- P：暂停
- M：总静音
- 右上角设置：音乐、音效、震动与界面抖动

浏览器会阻止页面自动播放声音，点击“启动战机”后配乐才会开始。震动功能取决于设备与浏览器是否支持 Web Vibration API。
